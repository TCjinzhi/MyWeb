#说明#
***
##环境说明##
 - windows 7
 - jdk 1.8
 - eclipse 4.6
 - maven 3.3.9
 - tomcat 8.0
 - mysql 5.7.12

##功能说明##
 * springMVC+mybatis架构
 * bootstrap、AmazeUI、umeditor等前端框架应用
 * websocket 多人聊天室
 * ajax异步刷新
 * mybatis分页工具PageHelper
 * cxf实现WebService  Soap & Restful
 * 使用druid内置的监控
 * aspect注解方式，整个项目中加入log日志
 * 单开线程读取第三方api数据
 * 集成redis，本地虚拟机做redis，java使用jedis调用
